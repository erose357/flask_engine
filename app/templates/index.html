Flask Engine API

Introduction

Flask Engine is the Turing School's Rales Engine project written in Python using Flask, SQLAlchemy, and Postgresql

The code for the project can be found on github at: flask_engine

Please also see Flask Engine Sales a project consuming data from this API

Available Endpoints

all endpoints return JSON

base URL: https://flask-engine-api/heroku.com

Merchants

GET /api/v1/merchants - returns all merchants

GET /api/v1/merchants/:id - returns a merchant object for the :id that was passed in

GET /api/v1/merchants/:id/invoices - returns a merchant object for the :id that was passed in along with all related invoices

GET /api/v1/merchants/:id/items - returns a merchant object for the :id that was passed in along with all related items

GET /api/v1/merchants/random - returns a random merchant object

Customers

GET /api/v1/customers - returns all customers

GET /api/v1/customers/:id - returns a customer object for the :id that was passed in

GET /api/v1/customers/:id/invoices - returns a customer object for the :id that was passed in along with all related invoices

GET /api/v1/customers/random - returns a random customer object
